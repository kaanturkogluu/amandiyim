1. Genel Bakış

Bu sistem, müşteriler, mağazalar ve yöneticiler için çok kullanıcılı bir e-ticaret platformudur. Sistem üç ana kullanıcı tipini destekler:

  • Müşteriler (Google ile giriş yapabilir)
  • Mağazalar (E-posta/şifre ile giriş)
  • Yöneticiler (E-posta/şifre ile giriş)

2. Sistem Mimarisi

2.1 Teknoloji Stack

  • Backend: PHP
  • Frontend: HTML, CSS, JavaScript
  • Veritabanı: MySQL
  • Kimlik Doğrulama: Google OAuth2

2.2 Mimari Yapı

Sistem MVC (Model-View-Controller) mimarisini kullanır:

  • Model: Veritabanı işlemleri ve iş mantığı
  • View: Kullanıcı arayüzü ve görünümler
  • Controller: İstek yönetimi ve iş akışı kontrolü

3. Klasör Yapısı

3.1 Ana Dizinler

Klasör Açıklama
/api API endpoint'leri ve işleyicileri
/assets Statik dosyalar (CSS, JS, resimler)
/classes PHP sınıf dosyaları
/config Yapılandırma dosyaları
/controller Controller sınıfları
/includes Ortak PHP dosyaları
/Model Veritabanı model sınıfları
/pages Frontend sayfa şablonları
/panel Yönetim paneli dosyaları
/uploads Kullanıcı yüklemeleri

5. Kullanıcı Yönetimi

5.1 Kullanıcı Tipleri

  • Müşteriler:
    • Google ile giriş yapabilir
    • Mağaza ürünlerini görüntüleyebilir
    • Sipariş verebilir
  • Mağazalar:
    • E-posta/şifre ile giriş
    • Ürün yönetimi
    • Sipariş yönetimi
    • Mağaza ayarları
  • Yöneticiler:
    • E-posta/şifre ile giriş
    • Mağaza yönetimi
    • Kullanıcı yönetimi
    • Sistem ayarları

5.2 Kimlik Doğrulama

Sistem iki farklı kimlik doğrulama yöntemi kullanır:

  • Google OAuth2 (Müşteriler için)
  • E-posta/Şifre (Mağazalar ve Yöneticiler için)

6. Mağaza Yönetimi

6.1 Mağaza Özellikleri

  • Mağaza profili yönetimi
  • Ürün kataloğu yönetimi
  • Sipariş takibi
  • Çalışma saatleri ayarları
  • Konum bilgisi yönetimi

6.2 Mağaza Ayarları

  • Mağaza logosu ve ana görsel
  • İletişim bilgileri
  • Adres ve konum bilgileri
  • Çalışma saatleri
  • Kategori ayarları

8. Güvenlik

8.1 Güvenlik Önlemleri

  • CSRF token koruması
  • XSS koruması
  • SQL injection koruması
  • Şifre hashleme (password_hash)
  • Session yönetimi

8.2 Dosya Güvenliği

  • Yüklenen dosyaların tür kontrolü
  • Dosya boyutu sınırlaması
  • Güvenli dosya isimlendirme

9. Frontend Yapısı

9.1 Kullanılan Teknolojiler

  • HTML5
  • CSS3
  • JavaScript (Vanilla)
  • Font Awesome (ikonlar)

9.2 Sayfa Yapısı

  • Responsive tasarım
  • Modüler CSS yapısı
  • Dinamik form validasyonu
  • AJAX tabanlı işlemler

10. Backend Yapısı

10.1 PHP Sınıfları

  • MyLogin: Kullanıcı girişi yönetimi
  • Stores: Mağaza işlemleri
  • Helper: Yardımcı fonksiyonlar
  • Session: Oturum yönetimi
  • CsrfToken: CSRF koruması

10.2 Controller Yapısı

  • loginController: Giriş işlemleri
  • storeController: Mağaza işlemleri
  • googleloginController: Google girişi
